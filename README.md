# Green Wizard Project: Sustainability & Onchain Economics
**Lead Analyst: Fidelis Musau Musyoka**

## Project Mission
Bridging the gap between **Sustainable Economic Theory** and **IT Infrastructure**. This project utilizes onchain data (Ethereum/Solana) to track capital flows into green assets, providing a data-driven framework for resource optimization.

## The Hybrid Framework
- **Economic Intelligence:** Cost-benefit analysis of decentralized green finance.
- **Data Engineering:** Utilizing SQL and Python to model liquidity shifts in eco-tokens.
- **IT Strategy:** Implementing secure, remote-first data pipelines for global transparency.

## Tech Stack
- **Languages:** Python (Pandas), SQL
- **Blockchain:** Dune Analytics, dbt (Data Build Tool)
- **Security:** VDI/VPN Network Architecture
